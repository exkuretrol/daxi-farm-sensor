### 描述

這是一個位於大溪農場的感測器視覺化網站，感測器主要分成兩個位置：

- 溫室
- 室外

感測器用於監測蘆筍的生長條件。

### 變數單位

| 變數         | 單位                        | 精度                                  |
| ------------ | --------------------------- | ------------------------------------- |
| 氣壓         | 帕                          | 0.06hPa                               |
| 氣溫         | °C                          | 0.5°C                                 |
| 空氣相對溼度 | %                           | 2%                                    |
| 光強度       | lux                         | 1lux                                  |
| 風向         | 對北方的相對角度            |                                       |
| 風速         | 伏特；需要乘以 6 m/s 0.1m/s |
| 土壤溫度     | °C                          | ±0.5°C(25°C)                          |
| 土壤濕度     | %                           | 0-53%範圍內為 ±3%;53-100%範圍內為 ±5% |
| 土壤電導度   | us/cm                       | 10us/cm                               |
